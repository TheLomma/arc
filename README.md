# ARC Twix v1.0

Live Map Conditions Tracker fuer ARC Raiders.
Stack: React + Vite + Tailwind CSS

## Setup
npm install
npm run dev

## Vercel
1. GitHub Repo pushen
2. vercel.com -> Import
3. Framework: Vite | Build: npm run build | Out: dist
4. Deploy!

icon-192.png + icon-512.png in /public/ ablegen.
